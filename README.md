Prerequisites:

- [Vercel CLI](https://vercel.com/docs/cli) installed globally

To develop locally:

```
npm install
npm run dev
```

```
open http://localhost:3000
```

Pass a different port to Vite when needed, for example `npm run dev -- --port 4000`.

To build locally:

```
npm install
vc build
```

To deploy:

```
npm install
vc deploy
```
