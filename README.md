# Shipping Binaries

A minimal [Hono](https://hono.dev/) website for Vercel.

## Development

Prerequisites:

- [Vercel CLI](https://vercel.com/docs/cli) installed globally

```sh
npm install
npm run dev
```

Open <http://localhost:3000>.

## Verification

```sh
npm run typecheck
vc build
```

## Deployment

```sh
vc deploy
```
